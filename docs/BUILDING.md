# Building Mise (0.0.1)

## Requirements

- Python 3.10+ (same version on macOS and Windows)
- Git
- (Optional) Virtualenv support

## Clone and run from source (macOS / Windows)

```bash
git clone https://github.com/you/mise.git
cd mise
python -m venv .venv
# macOS
source .venv/bin/activate
# Windows
# .venv\Scripts\activate
pip install .
mise

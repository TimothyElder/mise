# Changelog

## [0.1.0] — 2025-11-25
- First public alpha release of Mise.
- Basic project creation, document upload, coding, analysis, and report generation.

## [0.1.1] — 2025-11-26
- Bug fixes that address bad paths when project directory is moved. Most edits to ProjectRepository and AppController
- Needs more thorough audit and refactoring of how paths and doc ids are handled when calling data between classes.

## [0.1.2] — 2025-11-30
- UI updates including new default fonts. Noto Serif for document viewers and Noto Sans for all other text.
- Increase text size in document viewers now supported.
- Keyboard shortcuts and help dialog describing them added.
- Windows and macOS now supported.
